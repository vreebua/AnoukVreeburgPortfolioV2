To connect a physical object to a digital solution a concept was born. I made an easel with a projector that is controllable by voice, so you can simply change the canvas when you are busy with painting. This method is fast, makes sure your painted nails don't stain other things and it is overall fun and easy!

Virtual Reality

My first introduction with Virtual Reality was a real fun one! I came up with a concept that helps people with motionsickness that VR glasses can cause. I achieved this by first placing the user in a recognisable room, like a living room, before the real game starts. This way, the user will less hardly move into a hard fantasy VR world, by first adjusting the eyes through a fimiliar scene. All of this is also backed up by research!
